## Installation

#### Clone the project

```bash
  git clone https://github.com/AliNikandish/MusicPlayer.git
```

#### Go to the project directory

```bash
  cd MusicPlayer
```

#### Install QuizApp with npm

```bash
  npm install
```
#### Run:
```bash
  npm run dev
```
